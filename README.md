# Frontend Mentor - E-commerce product page solution

This is a solution to the [E-commerce product page challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/ecommerce-product-page-UPsZ9MJp6). Frontend Mentor challenges help you improve your coding skills by building realistic projects.

A responsive Frontend Mentor challenge built with **React**, **Vite**, and **SCSS**.

## Tech Stack
- React (hooks)
- Vite
- SCSS (Sass)
- JavaScript (ES6+)

## Features
- Product gallery with active thumbnails
- Desktop lightbox (prev/next + thumbnail switch)
- Cart dropdown (empty/filled states)
- Quantity selector and add-to-cart logic
- Mobile burger menu with overlay + drawer
- Fully responsive layout (desktop/mobile)
